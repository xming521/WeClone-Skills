<h3 align="center">🚀 A skill that helps your agent become your AI twin. 💡</h3>  

This repo separates the workflow into two skills: `init-twin` scaffolds the persona pack, and `twin-reply` uses an existing persona pack to draft review-gated digital twin replies.

Filled example persona packs and runtime scenes live in [examples/README.md](examples/README.md).
